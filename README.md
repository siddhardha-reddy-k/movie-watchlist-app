# 🎬 Movie Watchlist App

A full stack movie watchlist application built with the PERN stack.

## Features

- User authentication (Register/Login)
- Search movies via TMDB API
- Add/Remove movies from personal watchlist
- Protected routes
- Responsive dark UI

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS

**Backend:** Node.js, Express.js

**Database:** PostgreSQL (Neon)

**API:** TMDB API

## About This Project

This project was built while learning full stack development
with AI assistance. It helped me understand how React, Node.js,
Express, and PostgreSQL work together.

## What I Learned

- JWT authentication from scratch
- Building REST APIs with Express
- Connecting PostgreSQL with pg
- Protected routes in React
- Third party API integration (TMDB)

## Setup Instructions

### Backend

cd backend
npm install
create .env file (see .env.example)
node server.js

### Frontend

cd frontend
npm install
npm run dev
